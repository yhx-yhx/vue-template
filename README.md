# vue-template

## 介绍项目

基于vite实现快速开发脚手架

## 技术栈

- Vue3
- TypeScript
- Vite
- Pinia
- Vue Router
- Element Plus
- Axios

## 插件扩展

- autoprefixer
- code-inspector-plugin
- tailwindcss
- unplugin-auto-import
- unplugin-icons
- unplugin-vue-components
- @iconify/tailwind

## 项目结构

```
├── public
├── src
│   ├── assets
│   ├── components
│   ├── router
│   ├── store
│   ├── utils
│   ├── views
│   ├── App.vue
│   ├── main.js
├── .gitignore
├── index.html
├── package.json
├── README.md
├── tsconfig.json
├── vite.config.ts
```

## 安装依赖

```
npm install
```

### 启动服务

```
npm run dev
```

### 打包发布

```
npm run build
```
